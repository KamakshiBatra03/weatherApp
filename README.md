# 🌤️ SkyTune - Melodies for every shade of the sky

SkyTune is a dynamic weather web application that goes beyond basic forecasts. It matches the current weather conditions of any city with a custom curated musical playlist and motivational "Mood Advice," turning the daily weather report into an emotional vibe.

---

## ✨ Features

* **Real-time Weather Data:** Fetches instant temperature, humidity, weather conditions (e.g., Sunny), and wind speed for any city searched.
* **Mood Advice:** Gives personalized daily insights based on what the sky is doing (e.g., *"Let the sunshine fuel your creativity today."*).
* **Curated Mood Playlists:** Embedded dynamic playlists matching the atmospheric conditions (Sunny, Rainy, Windy, Cloudy) to match your mood to the skies.
* **Clean & Vibrant UI:** Features a sleek gradient design with smooth layout containers for optimal desktop and mobile viewing.

---

## 🛠️ Built With

* **HTML:** Semantic structure for layouts and components.
* **CSS:** Custom styles, beautiful color gradients, and glassmorphic card designs.
* **JavaScript :** Pure JavaScript split into modular components:
* * `script.js` / `code.js` - Manages global state and core API operations.
  * `mood.js` - Logic mapping weather conditions to emotional responses and advice.
  * `song.js` - Handles playlists and dynamic audio components.
